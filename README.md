# Huerto Virtual 

En esta aplicación podras plantar tu planta de tomate cherry desde tu Oculus Quest 2

## Instalación

Clonar el proyecto y abrir con la versión de Unity 2020.3.20f1

```bash
git clone https://github.com/catudo/HuertoVirtual.git
```
